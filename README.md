Vehicle Fleet Management System








A console-based Vehicle Fleet Management System developed in C++ using file handling and modular programming.
It allows efficient management of vehicles, trip scheduling, availability tracking, and trip history storage.

Features
Add new vehicles with validation
Edit existing vehicle details
Schedule trips with availability check
30-day vehicle availability calendar
View complete fleet status
Track trip history
Automatic fuel cost calculation
File-based persistent storage
Concepts Used
File Handling (fstream)
Arrays and Pointers
Functions
Input Validation
Modular Programming
Basic System Design

   Objective

This project demonstrates how a real-world fleet management system can be implemented using basic C++ concepts, focusing on file handling and data organization without using advanced libraries.
